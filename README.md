MERN To-Do App

A simple full-stack MERN (MongoDB, Express, React, Node.js) To-Do application that demonstrates CRUD functionality, real-time updates, and a responsive UI.

Features

Add, update, and delete tasks

Mark tasks as completed

Responsive React frontend

RESTful API backend with Express.js

MongoDB database for task storage

Ready for deployment (backend on Render, frontend on Vercel)

Technologies

Frontend: React, Axios, React Router

Backend: Node.js, Express.js, Mongoose

Database: MongoDB

Deployment: Render (backend), Vercel (frontend)

Getting Started (Local Development)
Prerequisites

Node.js and npm installed

MongoDB running locally or using MongoDB Atlas
